# Daemo Experiment

A simple agent equipped with a calculator.power() function, to test [Daemo's](https://docs.daemo.ai) engine.

## Setup

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Configure environment variables**

   Copy `.env.example` to `.env` and fill in your values:

   ```bash
   cp .env.example .env
   ```

3. **Run the application**
   ```bash
   npm run dev
   ```
